# Crimson's Unraid Templates

Custom XML templates for high-performance game servers and self-hosted tools.

## 🐉 RuneScape: Dragonwilds Dedicated Server
Optimized for the **0.11 Dowdun Reach** update.

### **Quick Setup**
1. **Owner ID:** You **must** enter your Player ID (found in-game settings) for the server to boot.
2. **Ports:** Ensure `7777` and `27015` (UDP) are forwarded on your router.
3. **RAM:** 8GB is recommended for a full 6-player session.

### **Manual Installation**
If not using Community Applications, add this repo URL to your Docker settings:
`https://github.com/Crimson7806/unraid-templates`
